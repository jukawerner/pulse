If creating visual artifacts (slides, mocks, throwaway prototypes, etc), copy assets out and create static HTML files for the user to view. If working on production code, you can copy assets and read the rules here to become an expert in designing with the Pulse brand.

Key points to remember:
- Pulse = trade marketing operations (promotores em campo, supervisores no dashboard, gestores em relatórios)
- Brand colors: red (#C8102E) for actions/CTAs, navy (#1B2A4E) for structure/data
- Typography: Space Grotesk (display), Manrope (body), JetBrains Mono (numeric/code)
- Voice: próximo e descontraído, "você" informal, frases curtas, sem jargão
- Density: spacious/modern, not data-heavy ERP-style
- Both light and dark themes are first-class — never assume light
