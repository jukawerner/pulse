# Pulse · Landing UI Kit

Landing institucional para captação B2B. Estrutura:

1. **Nav** com logo + CTA primário
2. **Hero** assimétrico: copy à esquerda, três cards de produto flutuantes à direita (formando uma "narrativa de dados")
3. **Trust strip** com logos placeholder de clientes
4. **Features** — grid 3×2, ícones em gradiente Pulse
5. **Band de números** — fundo navy-900, números grandes com linha de pulso decorativa
6. **Testimonial** centralizado, tipografia display generosa
7. **CTA final** — bloco gradient com pulse line decorativa
8. **Footer** padrão

## Notas
- Gradiente Pulse aparece em: ícones de feature, card "Pulse no ar" do hero, CTA final. **Nunca em tela cheia.**
- Hero usa imagem de "produto em ação" (três cards mock) ao invés de um screenshot — escala melhor sem precisar de assets reais.
- Logos de clientes são placeholders em texto. **Substituir por SVGs reais quando disponíveis.**
- Quote é fictícia — substituir por case real antes de publicar.
